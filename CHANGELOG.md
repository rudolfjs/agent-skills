## 0.2.0 - 2026-04-08
### Added
* Added solution spec for configurable pi-rpc session timeout
* Add Jules dispatch workflows for security review (@jules-security), issue triage and spec writing (@jules-review), and skill auditing/creation/fixing (@jules-skills).
* Add solution spec for pi-rpc provider mismatch and error masking issue (Issue
* Add Jules CI review dispatch workflow that automatically invokes Jules to diagnose and fix CI failures on pull request branches.
* Add lychee link checking as a pre-commit hook and CI workflow to catch broken URLs in skill files before they reach main.
* CONTRIBUTING.md with fork-based PR workflow and changie fragment requirement
* CI workflow to enforce changie unreleased fragments on PRs
* Pre-commit hooks for skill validation, ruff lint/format, and lock-file check
* Tag-triggered release workflow with automatic changie batch and GitHub release
* feat(gemini-cli): add gemini-3.1-flash-lite-preview to model selection
### Changed
* pi-rpc: add server-side default model/provider fallback (PI_DEFAULT_PROVIDER/PI_DEFAULT_MODEL), make CLI flags optional, and provide start.sh for simplified background startup
* Add MIT license field to all skill frontmatter
* promote GEMINI_SYSTEM_MD with .gemini/prompts/ convention
### Fixed
* CI validation failures and validator coverage for edge cases
* Prevent Python cache artifacts from being recommitted to the repository
* Release workflow failing to push changelog to protected main branch
* Release workflow tag drift, shallow fetch, and missing unreleased changes guard
* Release workflow using secrets instead of vars for public App ID
